Configuration Firebase

Pour activer la synchronisation en temps réel entre appareils, créez un fichier firebase-config.js dans le répertoire de l'application. 

export const firebaseConfig = {
  apiKey: "...",
  authDomain: "...",
  projectId: "...",
  storageBucket: "...",
  messagingSenderId: "...",
  appId: "..."
};
export const IS_CONFIGURED = true;

En attendant, l'application fonctionne en mode hors-ligne avec sauvegarde locale.
